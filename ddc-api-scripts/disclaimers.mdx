title: "Disclaimers"
---


The **GMG Dynamic Disclaimer Plugin** dynamically retrieves and injects
dealership-specific disclaimer content into:

-   Search Results Pages (SRP)
-   Vehicle Detail Pages (VDP)

Disclaimers are sourced from a centralized API and inserted cleanly into
Dealer.com pages.

------------------------------------------------------------------------

## Features

-   Works on both SRP and VDP
-   Automatically detects New vs Used inventory
-   Replaces existing disclaimer content if present
-   Inserts dynamically if no disclaimer exists
-   Optional placeholder support
-   24-hour caching via sessionStorage
-   Dealer-specific API-driven content
-   Uses Dealer.com Web Integration API lifecycle

------------------------------------------------------------------------

## Deployment Instructions

### Step 1: Add Script to Dealer.com

Navigate to:

Dealer.com Admin → Site Tools → Edit Custom Site JavaScript

Then:

1.  Open the "Bottom of Body" tab
2.  Paste the following script:

``` javascript
(function () {
  var s = document.createElement('script');
  s.src = 'https://assets.garberauto.com/assets/ddc-api/js/gmgDisclaimers.js';
  s.async = true;
  document.body.appendChild(s);
})();
```

3.  Enable the script block
4.  Save & publish

------------------------------------------------------------------------

## Important Notes

-   Do NOT place this in Head or Top of Body

## Placement Logic

The script injects disclaimers using the following priority order.

### 1. Replace Native Dealer.com Disclaimer

If the following element exists, the script replaces its content:

```html
<div data-widget-name="content-disclaimer"></div>
```

### 2. Use Custom Placeholder

If no native disclaimer is found, the script looks for this optional placeholder:

```html
<div id="gmgDisclaimerPlaceholder"></div>
```

### 3. Fallback Injection

If neither of the above exists, the script falls back to appending the disclaimer to the first available container:

- `.inventory-results`
- `.srp-listings`
- `.vehicle-list`
- `main .container`
- `main`

## Optional Placeholder Setup

To control disclaimer placement more precisely, add this HTML where you want the disclaimer to appear:

```html
<div id="gmgDisclaimerPlaceholder" class="ddc-content pt-3 text-muted"></div>
```
