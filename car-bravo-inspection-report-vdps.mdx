# Garber Vehicle Inspection & CarBravo Benefits Widget

**Version:** 1.0\
**Author:** Garber Digital Marketing Development Team

------------------------------------------------------------------------

## Overview

This script is a **Dealer.com Web Integration API widget** designed to
enhance Vehicle Detail Pages (VDPs).\
It dynamically inserts two UI components:

1.  **Vehicle Inspection Report Widget**
2.  **CarBravo Benefits Widget**

The widgets are rendered only when the vehicle meets the required
criteria.

------------------------------------------------------------------------

## Features

### Vehicle Inspection Widget

-   Displays **126‑Point Inspection status**
-   Pulls VIN-specific inspection report URLs from a **Google Sheets
    CSV**
-   Shows inspection categories in a **carousel UI**
-   Provides a **Download Full PDF Report** button
-   Uses animated PASS indicators for each category

### CarBravo Benefits Widget

Displays CarBravo program benefits including:

-   6‑Month / 6,000‑Mile Limited Warranty
-   10‑Day / 500‑Mile Exchange Program
-   Roadside Assistance
-   Vehicle History Report
-   Additional CarBravo protections

------------------------------------------------------------------------

## Page Conditions

The script runs only on:

-   `detailPage` (Dealer.com VDP)

Additional filters:

### CarBravo Filter

The script checks the vehicle's `optionCodes` field.

Example expected value:

    "optionCodes": [
      "CarBravo"
    ]

If the vehicle **does not include the CarBravo option code**, the
widgets will not render.

------------------------------------------------------------------------

## Data Source

Inspection reports are sourced from a **public Google Sheets CSV
export**.

Example:

    https://docs.google.com/spreadsheets/.../output=csv

Required CSV Columns:

  Column                  Description
  ----------------------- -------------------------------
  VIN                     Vehicle VIN
  Inspection Report URL   Link to PDF inspection report

------------------------------------------------------------------------

## Rendering Location

Widgets are injected into the Dealer.com layout using:

    API.insert('secondary-content')

This places the widgets within the **VDP secondary content column**.

------------------------------------------------------------------------

| Screen Size \| Cards Visible \|

\|-------------\|---------------\| Mobile \| 2 \| \| Tablet / Desktop \|
3 \|

-   Swipe (mobile)
-   Drag (desktop)
-   Arrow navigation

## Performance Optimizations

The script includes several optimizations:

-   **CSV caching** to prevent repeated downloads
-   VIN lookup performed once per page
-   CSS injected only once
-   Widget prevented from duplicate rendering

------------------------------------------------------------------------

## Script Flow

1.  Wait for `page-load-v1`
2.  Confirm page is `detailPage`
3.  Retrieve vehicle data
4.  Verify `optionCodes` contains `CarBravo`
5.  Fetch CSV inspection data
6.  Match VIN
7.  Insert widgets
8.  Initialize carousel UI

------------------------------------------------------------------------

## Encoding Recommendation

When editing documentation or CSV files use:

    Windows‑1252 (ANSI)

This prevents issues with characters like:

-   '
-   "
-   "

which can break CSV parsing if saved incorrectly.
